sedec

The sedec(=SEction coDEC) is module to encode/decode and also transcode section(ISO-13818).

Download this project as a .zip file Download this project as a tar.gz file

What is sedec?

The sedec(=SEction coDEC) is a library to encode/decode and also transcode section(ISO-13818),
Especially for AIT(=Application Information Table) of HbbTV 1.5/2.0.

The input of sedec is raw AIT table(Not section to be seperated)
And output is abstractive AIT object. User application can achieve all of information that AIT has.

How to build

Build tree are compose of followings:

.
├── CMakeLists.txt
├── LICENSE
├── README.md
├── build (Directory you will may create)
├── examples (This has samples of decoder/encoder)
├── release (Directory to release to cross platforms)
└── sedec(Sedec sources)

You can meet library(libsedec) in build/out directory after following build command.
And thus there are examples, e.g. decoder, encoder.

#mkdir build
#cd build
#cmake ..
#make

Encoder

Thie example encoder's real operation is to trascode from original AIT to modified AIT.
The way to run same as decoder example, but it has to check sources how example can transcode original AIT.

#encoder_example original_ait
#decoder_example modified_ait // you can see information if transcoding work out well.
